What it does

  • Renames ugly screenshots, downloads, folders, and AI-generated assets
  • Watches Downloads, Desktop, and Documents
  • Supports one-click rename, custom rename, batch rename, and Clean My Mess

Install

  1. Download RenamePolice-mac.zip
  2. Unzip it
  3. Move RenamePolice.app to Applications
  4. Open it and use the menu bar icon
